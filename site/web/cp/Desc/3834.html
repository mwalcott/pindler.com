3834-Fontainebleau is a Signature Exclusive Sunbrella indoor/outdoor pattern. Fontainebleau is a gorgeous large scale etchy damask pattern woven with performance Sunbrella 100% solution dyed all-weather acrylic yarns. This pattern has weave texture irregularities that were intentionally designed into the fabric that add a hand-woven, vintage appearance to it. Richard Frinier created these understated patterns to work for a variety of applications to evoke the romantic elegance of Provence.
