3835-Fleur de Lis is a Signature Exclusive Sunbrella indoor/outdoor pattern. Fleur de Lis is a beautiful mid scale etchy fleur de lis pattern woven with performance Sunbrella 100% solution dyed all-weather acrylic yarns. This pattern has weave texture irregularities that were intentionally designed into the fabric that add a hand-woven, vintage appearance to it. Richard Frinier created these understated patterns to work for a variety of applications to evoke the romantic elegance of Provence.
