4949-Lauder is an Exclusive upholstery indoor/outdoor geometric midscale pattern. Lauder is a transitional midscale ikat trellis design that is woven in a pocketweave using new "Linen Look" outdoor yarn that was made using a special heathered solution-dyed polyester yarn. Lauder is woven using an indoor/outdoor heathered yarn and is mildew, fade, soil and stain resistant.
