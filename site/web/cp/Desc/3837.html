3837-La Mer is a Signature Exclusive Sunbrella indoor/outdoor stripe. La Mer is a classic mid scale stripe woven with performance Sunbrella 100% solution dyed all-weather acrylic yarns. This pattern has weave texture irregularities that were intentionally designed into the fabric that add a hand-woven, vintage appearance to it. Richard Frinier created these understated patterns to work for a variety of applications to evoke the romantic elegance of Provence.
