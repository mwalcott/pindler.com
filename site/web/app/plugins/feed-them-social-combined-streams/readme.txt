Feed Them Social Combined Streams

== Description ==
Combine Facebook, Instagram, Twitter, Pinterest and Youtube posts in one Feed. Also you can mix facebook feeds too. See all details and demo here. https://www.slickremix.com/downloads/feed-them-social-combined-streams/

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
